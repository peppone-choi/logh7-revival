LOGH VII distribution layout

Client package:
- LOGH7Launcher.exe
- exe/G7MTClient.exe and legacy client data
- exe/D3D8.dll and exe/dgVoodoo.conf
- fonts/ and tools/packaging/install-pretendard.ps1

Server/admin package:
- logh7-runtime/src/server/
- logh7-runtime/content/
- logh7-runtime/state/accounts.sqlite
- logh7-runtime/state/world-state.sqlite
- logh7-runtime/logs/ and logh7-runtime/traces/

This local bundle keeps both sides together only for single-machine development smoke tests.
Release packaging must emit separate client and server archives; the player should not need the original LOGH VII installer.
