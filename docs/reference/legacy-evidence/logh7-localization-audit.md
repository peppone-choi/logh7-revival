# LOGH VII 한글화 감사 — 우선순위 워크리스트

> 본 문서는 4개 로컬라이제이션 소스에 대한 자동 감사 결과를 종합한 **워크리스트**다.
> 실제 번역 데이터를 고치는 작업 지시서가 아니라, 무엇이 깨졌고 무엇을 손대면 안 되는지 정리한 후보 목록이다.

## 운영 원칙 (반드시 준수)

1. **커밋된 번역 덮어쓰기는 라이브 검증 후에만.** 이 문서의 어떤 항목도 그 자체로 "고쳐라" 지시가 아니다. 실제 깨진 라벨을 수정·재추출하기 전에 반드시 실클라(`logh7-live` / `ui_explorer`)에서 해당 텍스트가 화면에 노출되는지, 어떻게 깨져 보이는지 확인한 뒤 적용한다.
2. **캐논 고유명사 임의 한글화 금지.** 함급명·세력명 등 원작 고유명사가 의심되는 항목은 별도 섹션으로 분리했다. 추정 한글명을 임의로 채워 넣지 말고, 원본 바이트 재추출 또는 캐논 소스(매뉴얼/전작 DB) 대조로 확정한 값만 반영한다.
3. **표시텍스트 vs 비표시 데이터 구분.** 욕설 필터(g7sw.dat)·폰트 페이스명·버전/저작권 문자열·printf 포맷·내부 센티넬은 화면 표시 텍스트가 아니므로 한글화 대상이 아니다(저우선 또는 비대상).
4. **모지바케의 진짜 원인.** 대부분의 "깨짐"은 미번역이 아니라, 추출 툴이 **cp949 한글을 cp932로 오판독**해 생긴 mojibake다. 원바이트(cp949) 재추출이 1차 해법이며, 일부는 바이트 손상으로 부분 복원만 가능하다.
