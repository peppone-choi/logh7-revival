## 3. 캐릭터 생성 흐름 (1920 풀스크린)
- `ui_explorer create-character --session-row N --faction empire|alliance --lastname X --firstname Y --flagship Z`
  명령이 21스텝 좌표를 자체 인코딩(세션행 더블클릭→진영→성별/출신→이름필드→next). 1920에서 **작동 확인**(재캘리브 불필요).
- 세부 좌표는 `tools/logh7_ui_explorer.py cmd_create_character` 참조(필요 시 추출).
