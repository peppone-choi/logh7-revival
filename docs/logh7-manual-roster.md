# LOGH VII — Manual Roster (Game-Start Duty-Card Holders)

Source: gin7 official manual (`gin7manual.txt`), 101p, archive.org/gin7manual. Extracted 2026-06-12. Machine-readable companion: `content/roster/manual-roster.json`.

> **IMPORTANT — these are game-START ranks, not end-state.** At scenario start the 門閥貴族 (highborn-noble) era is still in force: Reinhard von Lohengramm is only a 中将 (Vice Admiral) commanding the 2nd Fleet, and Siegfried Kircheis is a 少佐 (Lt. Commander) serving as his adjutant. These are NOT the late-series flag-officer stats. The existing in-code `NAMED_CANON` end-state stats (and `content/roster/canon-extra.json`) describe much later career states and MUST be reconciled against this start-state table before either is treated as authoritative for a given scenario epoch.

## Abilities (能力値 — 8 stats)

| # | Name | Effect |
| --- | --- | --- |
| 1 | 統率 | 人材活用能力。要職就任時は徴税額・政府支持率に影響。艦隊司令官時は艦隊最大士気・降伏勧告成功率に影響 |
| 2 | 政治 | 政治力。高いほど市民の支持を集めやすい |
| 3 | 運営 | 事務管理能力。惑星の統治などに影響 |
| 4 | 情報 | 情報収集・分析能力。スパイ活動や戦術における索敵能力に影響 |
| 5 | 指揮 | 部隊指揮能力。高いほど配下の艦船を迅速に行動させられる |
| 6 | 機動 | 艦隊移動に関する指揮能力。高いほど機敏な操艦が可能 |
| 7 | 攻撃 | 攻撃に関する指揮能力。艦隊司令官/幕僚時は艦隊の攻撃能力、惑星防衛司令官時は惑星の対艦防衛能力に影響 |
| 8 | 防御 | 防御に関する指揮能力。艦隊司令官/幕僚時は艦隊の防御能力、防衛司令官時は敵艦隊の惑星占領に対する惑星防御能力に影響 |
